HTML
HTML (HyperText Markup Language) é a linguagem de marcação padrão utilizada para criar e estruturar páginas web.
CSS
CSS (Cascading Style Sheets, ou Folhas de Estilo em Cascata) é a linguagem de estilos utilizada para definir a aparência, o layout e a formatação de documentos HTML, transformando o conteúdo estruturado em uma página visualmente organizada e atraente.
Javascript
JavaScript é uma linguagem de programação de alto nível, leve e interpretada, usada principalmente para criar interatividade e conteúdo dinâmico em páginas web.
